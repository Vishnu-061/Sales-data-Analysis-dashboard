# Sales-data-Analysis-dashboard
Analyzed sales dataset using Python, SQL, and Power BI to identify trends, perform data cleaning, and create visualizations. Developed an interactive dashboard to present sales performance, profit analysis, and regional insights.


Sales Data Analysis Dashboard
Overview
        This project analyzes sales data using Python, SQL, Excel, and Power BI.
Tools Used
        Python
        Pandas
        NumPy
        Matplotlib
        SQL
        Excel
        Power BI
Folder Structure
        dataset → raw dataset
        python → analysis code
        sql → SQL queries
        excel → cleaned data
        powerbi → dashboard file
Instructions
        Place it in dataset folder
        Run python/sales_analysis.py
        Create Power BI dashboard
